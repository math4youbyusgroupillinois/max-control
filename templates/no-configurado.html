Panel de control de Servidor de Centro

MAX Control no está configurado.

  1. Dentro de EBOX crea un usuario con permisos de administrador.
    (sería recomendable que no uses este usuario para nada más)


  2. Desde EBOX crea el grupo 'Teachers'.
    Desde 'MAX Control' se podrán añadir usuarios a este grupo (o desde EBOX).


  3. Desde EBOX crea un nuevo Directorio compartido de nombre 'isos'
    Edita el control de acceso y añade el grupo profesores 'Teachers' como lectura y escritura.


  4. Desde EBOX ver la contraseña de superadministración Menú ->Office -> Datos LDAP


  5. Editar el archivo /etc/max-control/conf.inc.php:


    • Cambiar define("CONFIGURED", False); => por define("CONFIGURED", True);


    • Editar define("LDAP_ADMIN", 'admin') y define("LDAP_PASS", 'EDITAME') con los datos del punto (1)


    • Editar define("LDAP_BINDPW", 'EDITAME') con la contraseña del punto (4)


  6. Vuelve a esta página y podrás entrar con cualquier usuario del dominio
