Panel de control de Servidor de Centro

MAX Control no está configurado.

  1. Dentro de EBox / Zentyal cree un usuario con permisos de administrador.
    (luego se puede borrar ese usuario desde max-control)


  2. Dentro de EBox / Zentyal activar los módulos:


    • Red
    • DHCP
    • Compartir ficheros
    • Usuarios y grupos
    • MAX Control


  3. Reinicie el servidor

  4. Ejecute desde una consola local o por SSH:

    sudo configure-max-control

  5. Vuelva a esta página y podrá entrar con cualquier usuario del dominio
